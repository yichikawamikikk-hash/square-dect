# square HP ニュース記事の追加手順

記事の追加は **news.json を編集** し、**写真を images フォルダに置く** だけで完了します。
HTMLファイル（index.html / news.html）は一切触る必要はありません。

---

## ファイル構成

```
リポジトリ直下/
├── index.html          ← トップページ（最新3件を自動表示。編集不要）
├── news.html           ← 記事ページのテンプレート（編集不要）
├── news.json           ← 記事データ。記事を足すときはここを編集
└── news/               ← 記事用の写真を入れるフォルダ
    ├── sample-002/
    │   ├── thumb.jpg    ← 一覧・記事トップに出るメイン写真
    │   ├── photo1.jpg   ← 本文下のギャラリー写真
    │   └── photo2.jpg
    └── （記事ごとにフォルダを作る）
```

---

## 記事を1本追加する手順

### 手順1：写真を用意してアップロード

1. `news/` の中に、記事用のフォルダを作る（例：`news/aktio-2026/`）
   - フォルダ名は半角英数字とハイフンで。スペースや日本語は避ける
2. その中に写真を入れる
   - `thumb.jpg` … 一覧と記事トップに表示されるメイン写真（必須でなくても可）
   - `photo1.jpg`, `photo2.jpg` … 本文の下に並ぶ写真（何枚でもOK。不要なら無し）

### 手順2：news.json に記事を1ブロック追記

`news.json` を開き、**いちばん上の `[` のすぐ下** に、以下のブロックを貼り付けます。
（新しい記事を上に足すと管理しやすいですが、順番は日付で自動並び替えされるのでどこでもOK）

```json
  {
    "id": "aktio-2026",
    "date": "2026-07-01",
    "title": "アクティオ様の建設現場に square を導入いただきました",
    "thumb": "news/aktio-2026/thumb.jpg",
    "images": [
      "news/aktio-2026/photo1.jpg",
      "news/aktio-2026/photo2.jpg"
    ],
    "body": "本文の1段落目をここに書きます。\n\n空行（\\n\\n）で区切ると段落が分かれます。\n\n3段落目もこのように続けて書けます。"
  },
```

**各項目の意味**

| 項目 | 内容 | 注意 |
|------|------|------|
| `id` | 記事の固有ID。URLになる | 他の記事と重複しないこと。半角英数字とハイフン |
| `date` | 日付 | 必ず `YYYY-MM-DD` 形式（例 2026-07-01）。新しい順に自動で並ぶ |
| `title` | 記事タイトル | |
| `thumb` | メイン写真のパス | 写真が無ければ `""`（空）でOK |
| `images` | ギャラリー写真のパスの配列 | 写真が無ければ `[]`（空）でOK |
| `body` | 本文 | 段落を分けたい所は `\n\n` を入れる |

### 手順3：GitHubに保存（コミット）

`news.json` と `news/` の写真をアップロード（コミット）すれば、数十秒〜数分で公開サイトに反映されます。

---

## よくある注意点（エラーを防ぐ）

- **記事と記事の間は必ずカンマ `,` で区切る。** ただし **最後の記事の後ろにはカンマを付けない**。
  ```json
  [
    { ...記事A... },   ← カンマあり
    { ...記事B... }    ← 最後はカンマなし
  ]
  ```
- **記号は半角で。** `{ } [ ] " : ,` はすべて半角。全角になっているとエラーで全記事が表示されなくなります。
- **本文中で " (ダブルクォート) を使いたいときは** `\"` と書く（バックスラッシュを付ける）。
- 不安なときは、編集後に [JSONLint](https://jsonlint.com/) などのチェックサイトに news.json を貼り付けると、書式ミスを見つけられます。

## 表示の仕組み（参考）

- トップページ（index.html）の News欄には、**日付の新しい順に最新3件** が自動表示されます。
- 記事が4件以上になると、トップに「すべてのお知らせを見る」ボタンが自動で出ます。
- 各記事は `news.html?id=（記事のid）` というURLで個別ページになります。
- `news.html`（idなし）にアクセスすると、全記事の一覧が表示されます。
- ページはパスワード保護されており、トップで一度入力すれば同じタブ内では記事ページも再入力不要です。
